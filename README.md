# reception-app

  会社用の受付アプリです。
  管理画面から予定を登録し、来訪者用画面にて予定を確認。
  その後予定に紐付いた担当者を呼び出すことができます。
  
  例としてChatWorkで来訪者の通知を送信しています。
      
## Installation
  # create database recep
  mysql -u root recep < dump.sql
